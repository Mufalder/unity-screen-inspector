Place "ScreenInspector" on empty GameObject or any other GameObject. Configure the script according to this description:

Show - show the screen limitations.
Mirrored - mirror the screen limitations.
Color - color of the screen limitations.

Corners - draw corners on the screen.
Type - type of screen limitations: none, iphone, waterdrop, punch hole.

HomeButton - draw the homebutton.
AligType - punch hole position.

Screen limitations automatically hides after entering the "Game mode".

Enjoy!
